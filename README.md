# z-ai-prompts

This repository contains various AI prompts and a command-line interface (CLI) to interact with them.

## Usage

The CLI tool `z-ai-prompts` can be executed using `npx` (if the package is published) or by running the `bin/cli.sh` script directly.

### Running with `npx` (if published)

```bash
npx z-ai-prompts <prompt-name>
```

## Available Prompts

The following prompts are available:

- `chouju-giga`
- `ready-markdown-lint`
- `japan-fiber`
- `refine-system-prompt`
- `ocr`
- `github`
- `japanese-chinese-translation`
- `enhance-photo`
- `generate-icon`
- `create-tool`
- `refine-markdown`
- `research-product-chinese`

## Example

```
> npx z-ai-prompts japanese-chinese-translation '我認為這個工具不錯, 可以測試一下.'
このツールは良いと思います。試してみましょう。
```
